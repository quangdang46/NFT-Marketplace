## Form Data Properties

### 1. Collection Details

- `chain` - The selected blockchain network (e.g., "base", "ethereum", "arbitrum")
- `name` - The name of the NFT collection
- `symbol` - The token symbol for the collection
- `description` - The description of the collection
- `collectionImage` - The uploaded collection image file


### 2. NFT Art Type

- `artType` - The type of NFT art ("same" or "unique")
- For "same" art type:

- `artwork` - The uploaded artwork file



- For "unique" art type:

- `metadataUrl` - The URL to the metadata (e.g., "[https://ipfs.io/ipfs/](https://ipfs.io/ipfs/)`<CID>`")





### 3. Mint Details

#### General Settings

- `mintPrice` - The base price for minting an NFT
- `royaltyFee` - The percentage fee for royalties
- `maxSupply` - The maximum number of NFTs that can be minted
- `mintLimit` - The maximum number of NFTs a single wallet can mint
- `mintStartDate` - The start date and time for the mint


#### Allowlist Stages

For each allowlist stage:

- `id` - Unique identifier for the stage
- `mintPrice` - The price for minting during this stage
- `durationDays` - The duration of this stage in days
- `durationHours` - The duration of this stage in hours
- `wallets` - The list of wallet addresses allowed to mint during this stage


#### Public Mint

- `mintPrice` - The price for public minting
- `durationDays` - The duration of the public mint in days
- `durationHours` - The duration of the public mint in hours
